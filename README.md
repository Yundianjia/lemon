## 容器服务器

选用的 Jetty 而不是 Tomcat。 原本, 以为想到一个不错的想法, 结果发现开源的 OA 系统, 并不是一个极其好的想法。

## 使用

利用 `mvn` 将项目启动后, 需要修改 `pom.xml` 中关于启动端口的配置。 项目中,默认使用的是 h2 数据库,
